flash中国地图

引用：http://img1.cache.netease.com/f2e/lib/js/ne.js

引用：http://qa.developer.163.com/component/flashmap/js/flashmap.js

概述：

中国地图flash组件，可以配置大小、颜色、透明度等。鼠标操作会通过回调通知。

语法：

var map = flashMap(config, data, callback);

参数：

  • config
    名称 类型 默认值 描述
    swf string null swf地址
    targetId string null 替换元素的id
    width string null 地图宽，高度自适应
  • data
    名称 类型 默认值 描述
    fillcolor string #3881C2 初始填充色
    fillalpha number 1 初始透明度
    overcolor string #FF0000 鼠标移入颜色
    overalpha number 1 鼠标移入透明度
    downcolor string #5D5153 鼠标按下颜色
    downalpha number 1 鼠标按下透明度
    bordercolor string #FFFFFF 边界填充色
    borderalpha number 1 边界透明度
    borderthickness number 1 边界粗细
    labelenable boolean true 是否显示地区名称
    labelfontsize number 12 地区名称字体大小
    labelcolor string #000000 地区名称颜色
    labelbgColor string #FFFFFF 地区名称背景颜色
    labelbgAlpha number 1 地区名称背景透明度
    toggle boolean false 地区鼠标按下是否弹起
    radio boolean false 按下一个地区，其它地区是否弹起(toggle为true时有效)
    shadow boolean true 是否显示阴影
    initColor object null 初始化时传入某些地区特殊颜色 initColor: {"110000": {"fillcolor":"#FF5F10", "overcolor":"#EEEEEE"}}
  • callBack
    名称 类型 默认值 描述
    positionData function 初始化后调用，返回object，包括各省位置信息
    click function 点击时回调，返回object，包括点击发生的位置信息
    hover function 移入时回调，返回object，包括移入发生的位置信息
    out function 移出时调用，返回object，包括移出发生的位置信息

例子：

//替换页面上ID为chinaMap的元素，设置宽为600，配置颜色；注册点击时回调
var map = flashMap({
    swf: "mapofchina.swf",              
    targetId: "chinaMap",
    width: "600"
  },{
    fillcolor: "#ED752D",
    overcolor: "#FB9D64",
    downcolor: "#6F6F6F"
  },
  {
    click: function(data) {
      data.code//点击区域代码
      data.name//点击区域名
      data.position//点击区域位置
      data.selected//点击区域是否被选中
    }
  }
);

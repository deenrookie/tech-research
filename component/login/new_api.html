NTESCommonLogin.login

概述：

实现登录验证功能。

语法：

NTESCommonLogin.login(authData, callback(userInfo));

参数：

  • authData
    名称 类型 默认值 描述
    email string null 通行证账号
    password string null 密码
    rememberMe boolean false 是否记录登录状态
  • userInfo
    名称 类型 默认值 描述
    status string 验证返回状态码
    msg string 验证返回消息
    name string 通行证‘@’之前的部份
    email string 通行证账号
    ip string 登录ip

例子：

//点击btn，验证登录
btn.click = function () {
  NTESCommonLogin.login({
    email: "xxx@163.com",//通行证
    password: "xxxxxx",   //密码
    rememberMe: true     //记住我
  }, function(userInfo) {
    userInfo.status         //验证返回状态码
    userInfo.msg            //验证返回消息
    userInfo.name         //通行证‘@’之前的部份
    userInfo.email         //电邮
    userInfo.ip              //ip
  });
};

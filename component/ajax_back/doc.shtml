ajax_back

其实严格意义上，应该是javascript执行的回退方案。通过hashchange来管理每一步js执行逻辑。

引入文件 http://qa.developer.163.com/component/ajax_back/ajax_back.js

代码实例

	var _back = new ajax_back();
		_back.init();
		_back.execute(
			function(){
				//需要回退的执行逻辑
			},
			hash//对应hash值，可缺省,会随机生成一个hash
		);

Emoji

融云消息 message 里的 emoji 说明

  1. 发消息时, 必须直接发送emoji原生字符. 如：😀 , 转换方法：symbolToEmoji

  2. Web SDK接收消息时接收到的是unicode编码格式, 如：”ef600”.需要使用messageDecode方法转化成原生emoji

SDK 说明

  1. emoji SDK 内置了128个emoji表情的图片, 主要是为了做消息输入框的表情选项, 也可自行扩展配置.

  2. SDK 初始化

        /* 1, 使用默认配置, 直接初始化 */
    RongIMLib.RongIMEmoji.init();

    /* 2, 不使用默认配置, 自定义初始化 */
   
    // 表情信息可参考 http://unicode.org/emoji/charts/full-emoji-list.html
    
    // 自定义设置
    var config = {
        size: '大小',
        url: '所有emoji的背景地址',
        lang: '选择的语言',
        reg: '匹配unicode的正则表达式'
    };
    // 扩展表情, 设置扩展表情的信息
    var emoji = {
        dataSource: {
            "表情unicode码":{
                "en":"英文名称",
                "zh":"中文名称",
                "tag":"原生表情字符",
                "position":"表情对应背景图的坐标"
            },
        },
        url: "新增emoji背景图片地址"
    };
    // 初始化
    RongIMLib.RongIMEmoji.init(config, emoji);

  3. Emoji SDK API

    方法 使用示例 方法说明
    names var names = RongIMEmoji.names; 获取全部表情的对应文字说明
    ["[大笑]", "[露齿而笑]", ………]
    emojis var emojis = RongIMEmoji.emojis; 获取全部表情的html标签
    [span, span, ………]
    nativeEmojis var emojis = RongIMEmoji.nativeEmojis; 获取全部原生emoji字符
    ["😀", "😁", ………]
    data var data = emoji.data; 获取所有数据，方便二次开发
    [{emoji:"😀",en:"grinning",name:"[大笑]",zh:"大笑"}, ………]
    emojiToSymbol(emoji, reg) var symbol = RongIMEmoji.emojiToSymbol('我是一个表情😁');
    返回值: 我是一个表情[露齿而笑]    		 将原生emoji转化为对应文字
    symbolToEmoji(symbol) var emoji = RongIMEmoji.symbolToEmoji('我是一个表情[露齿而笑]');
    返回值: 我是一个表情😁    		 将emoji的对应名称转化为原生emoji字符
    emojiToHTML(emoji, size, reg) var html = RongIMEmoji.emojiToHTML('我是一个表情😁'); 将原生emoji转化为span标签, 默认标签大小为24px
    symbolToHTML(symbol, size, reg) var html = RongIMEmoji.symbolToHTML('我是一个表情[露齿而笑]') 将原生emoji转化为span标签
    messageDecode(emoji, reg) var message = RongIMEmoji.messageDecode('收到一条消息 \uf601');
    返回值： 收到一条消息 😁    		 将emoji的unicode码转化为对应的原生emoji字符
    isSupportEmoji var isSupportEmoji = RongIMEmoji.isSupportEmoji; 判断是否支持emoji的渲染
    setupLanguage(lang) RongIMEmoji.setupLanguage('en'); 设置emoji文字的语言
